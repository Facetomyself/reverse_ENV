##  结论摘要

这个 POST body 的核心不是 RSA、ECC、AES 这类标准加密，也不是不可逆哈希。

它的实际流程是：

  *   *   *   *   *   *


    浏览器信号对象  -> JSON UTF-8 bytes  -> deflate-raw 压缩  -> 自定义随机 wrapper  -> 9 字节循环 XOR  -> fetch POST body

其中 wrapper 可逆，XOR 可逆，deflate-raw 可逆。因此，对这一次 trace 到的请求，可以完整还原出内层 JSON。

本次已AI自动还原出的明文 JSON+  离线纯算脚本。

###  6.5 为什么明文 JSON 不能只靠源码凭空复现

明文 JSON 本身是可读的、可解析的，但它不是纯常量。要生成同一次完全相同的 JSON，必须固定所有运行态输入：

  *   *   *   *   *   *   *   *


    1. 页面配置：apiKey、endpoint、agent script URL、SDK 版本2. session：linkedId、storage/cookie 中已有值3. 浏览器环境：UA、平台、语言、时区、窗口尺寸4. DOM/CSS：字体、系统颜色、文本测量、layout geometry5. 图形：Canvas/WebGL/像素摘要6. 时间：Date.now、performance.now、performance.timeOrigin7. 随机：crypto.randomUUID、crypto.getRandomValues、Math.random8. 异步时序：Promise、timer、CompressionStream、fetch 调度顺序

源码入口：

` of  ` 的纯算法等价于：

  *   *   *   *   *   *   *   *   *   *   *   *   *   *   *   *   *   *   *   *   *   *   *   *   *


    function of(payloadBytes, marker, maxPadding, keyLength, randomByte) {  const paddingLength = randomByte() % (maxPadding + 1);  const seed = randomByte();  const totalLength =    1 + marker.length + 1 + paddingLength + keyLength + payloadBytes.length;  const out = new Uint8Array(totalLength);  let offset = 0;  out[offset++] = seed;  for (const markerByte of marker) {    out[offset++] = (seed + markerByte) & 0xff;  }  out[offset++] = (seed + paddingLength) & 0xff;  for (let i = 0; i < paddingLength; i++) {    out[offset++] = randomByte();  }  const key = new Uint8Array(keyLength);  for (let i = 0; i < keyLength; i++) {    key[i] = randomByte();    out[offset++] = key[i];  }  for (let i = 0; i < payloadBytes.length; i++) {    out[offset++] = payloadBytes[i] ^ key[i % keyLength];  }  return out.buffer;}

    const Rr = new Uint8Array(1);function Cg() {  crypto.getRandomValues(Rr);  return Rr[0];}

##  本次随机值与 body 头部

DOM Trace 记录到  ` Cg -> of -> Vf  ` 下连续 13 次随机字节：

解释：

  *


    7e % 4 = 2

  *


    paddingLength = 2

  *   *   *


    seed = 3apadding = 79 8ckey = a3 e8 20 ea e3 c4 f9 9d bd

    3a 3d 48 3c 79 8c a3 e8 20 ea e3 c4 f9 9d bd

##  完整正向生成伪代码

下面是剥离浏览器采集后的纯算法版本。假设已经拿到  ` signalObject  ` ：

  *   *   *   *   *   *   *   *   *   *   *   *   *   *   *   *   *   *   *   *   *   *   *   *   *   *   *   *   *   *   *   *   *   *   *   *   *   *   *   *   *   *   *   *   *   *   *   *   *   *


    import { deflateRawSync } from "zlib";const QI = [3, 13];const ZI = [3, 14];const MAX_PADDING = 3;const KEY_LENGTH = 9;const COMPRESSION_THRESHOLD = 1024;function utf8JsonBytes(obj) {  return Buffer.from(JSON.stringify(obj), "utf8");}function randomByte() {  const b = new Uint8Array(1);  crypto.getRandomValues(b);  return b[0];}function wrap(payloadBytes, compressed) {  const marker = compressed ? ZI : QI;  const paddingLength = randomByte() % (MAX_PADDING + 1);  const seed = randomByte();  const padding = Buffer.alloc(paddingLength);  const key = Buffer.alloc(KEY_LENGTH);  for (let i = 0; i < paddingLength; i++) {    padding[i] = randomByte();  }  for (let i = 0; i < KEY_LENGTH; i++) {    key[i] = randomByte();  }  const out = Buffer.alloc(    1 + marker.length + 1 + paddingLength + KEY_LENGTH + payloadBytes.length  );  let offset = 0;  out[offset++] = seed;  for (const m of marker) {    out[offset++] = (seed + m) & 0xff;  }  out[offset++] = (seed + paddingLength) & 0xff;  padding.copy(out, offset);  offset += padding.length;  key.copy(out, offset);  offset += key.length;  for (let i = 0; i < payloadBytes.length; i++) {    out[offset + i] = payloadBytes[i] ^ key[i % KEY_LENGTH];  }  return out;}function buildPostBody(signalObject) {  const plain = utf8JsonBytes(signalObject);  const compressed = plain.length > COMPRESSION_THRESHOLD;  const payload = compressed ? deflateRawSync(plain) : plain;  return wrap(payload, compressed);}
