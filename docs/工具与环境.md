# 工具依赖与环境

## 核心约束

所有工具安装在 `D:\reverse_ENV\` 内，不污染系统全局环境。

## 目录速查

| 类别 | 路径 |
|------|------|
| Python venv | `D:\reverse_ENV\.venv\` |
| JDK | `D:\reverse_ENV\tools\jdk\` |
| Node.js | `D:\reverse_ENV\tools\node\` |

## 雷电模拟器

| 项目 | 值 |
|------|-----|
| 版本 | LDPlayer 9 |
| 安装路径 | `C:\Program Files\ldplayer9box\` |
| 后端 | VirtualBox 6.1.50 |
| VM 名称 | `leidian0` |
| ADB 地址 | `127.0.0.1:5555` |
| 管控脚本 | `tools\ldplayer\ldplayer.ps1` |

## 更新工具后

修改工具版本或路径后，必须同步更新：
1. 本文档
2. `CLAUDE.md` 的工具速查表
3. 相关 skill 的 fallback 路径
