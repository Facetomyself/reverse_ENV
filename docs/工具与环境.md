## First 微信小程序调试

| 项目 | 值 |
|------|-----|
| 版本 | 1.0.9 (开源版) |
| 仓库 | https://github.com/Spade-sec/First |
| 路径 | `tools\First\` |
| GUI 启动 | `powershell -File tools\First\first-gui.ps1` |
| CLI 启动 | `powershell -File tools\First\first-cli.ps1` |
| Python | `.venv\Scripts\python.exe` |
| 用途 | 微信小程序 Frida 注入 + CDP 代理调试 |
| 推荐微信 | 4.1.10 |

功能：Frida 注入微信 WMPF 进程 → CDP 代理桥接 Chrome DevTools → 云函数捕获、wx.* API 捕获、路由枚举导航、wxapkg 解密解包 + 敏感信息扫描、MCP Server 集成。

依赖：`frida>=17.0.0`, `websockets>=12.0`, `protobuf>=4.0.0`, `PySide6>=6.5.0`, `pycryptodome`（均已装在 venv）。

> **编译版存档**: v1.1.3 编译版（`First.exe` + `First-Windows.zip`）存于 `storage\First-release\`。当 WMPF 版本超过开源版覆盖范围时，用 `pyinstxtractor-ng` 提取新版 `addresses.{version}.json` 配置，复制到 `tools\First\frida\config\win\`。当前已额外补充 19977 / 19841 / 19871 / 19881 四个版本。

## reqable-mcp 抓包数据 MCP

| 项目 | 值 |
|------|-----|
| 版本 | 0.3.2 |
| 仓库 | https://github.com/ElonJask/reqable-mcp |
| 路径 | `tools\reqable-mcp\`（源码 + editable install） |
| CLI | `.venv\Scripts\reqable-mcp.exe mcp` |
| 用途 | 查询 Reqable 抓取的 HTTP/WebSocket 流量 — 请求搜索/域名统计/API 分析/代码生成 |

工作流：
1. 在 [Reqable](https://reqable.com) 桌面端配置 Report Server → `http://127.0.0.1:18765/report`
2. Reqable 抓包 → HAR 数据推送到 reqable-mcp
3. Claude Code 通过 MCP 工具查询分析流量（`reqable_*` 前缀，17 个 tools）

安装：`pip install -e tools\reqable-mcp`（已装在 venv，editable 模式）。

MCP 配置已写入 `.mcp.json`，前缀 `reqable_*`。
