# 脚本参考

所有脚本为 PowerShell `.ps1`，必须通过绝对路径调用：

```powershell
powershell -File "D:\reverse_ENV\skill\<name>\scripts\<script>.ps1" -参数名 值
```

禁止写临时 ad-hoc PowerShell 命令替代已有脚本。

## APK 逆向脚本

| 脚本 | 路径 | 用途 |
|------|------|------|
| `decode.ps1` | `skill/apk-reverse/scripts/` | jadx + apktool 一键落盘，产出摘要 |
| `frida-run.ps1` | `skill/apk-reverse/scripts/` | Frida 设备/进程管理、spawn/attach |
| `rebuild-sign-install.ps1` | `skill/apk-reverse/scripts/` | 重建 → 签名 → adb install |
| `manifest-summary.ps1` | `skill/apk-reverse/scripts/` | 抽取 Manifest 组件/权限 |

## IDA 逆向脚本

| 脚本 | 路径 | 用途 |
|------|------|------|
| `start.ps1` | `skill/ida-reverse/scripts/` | 验证环境就绪（不再是 HTTP 启动） |
| `open.ps1` | `skill/ida-reverse/scripts/` | idalib 打开文件（含 System32/锁文件绕过） |

## radare2 脚本

| 脚本 | 路径 | 用途 |
|------|------|------|
| `recon.ps1` | `skill/radare2/scripts/` | 一站式侦察：信息/节区/导入/导出/字符串 |

## 工具启动脚本

| 脚本 | 路径 | 用途 |
|------|------|------|
| `jadx-gui.cmd` | `tools\` | 启动 jadx-gui（自动设 JAVA_HOME 和加载插件） |
| `jadx.cmd` | `tools\` | jadx CLI 包装（自动设 JAVA_HOME） |
| `apktool.bat` | `tools\apktool\` | apktool 包装（自动设 JAVA_HOME） |
