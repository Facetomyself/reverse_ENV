# 脚本参考

## 调用规范

PowerShell 脚本 (`.ps1`)，绝对路径调用：

```powershell
powershell -File "D:\reverse_ENV\skill\<name>\scripts\<script>.ps1" -参数名 值
```

Bash 脚本 (`.sh`)，用 Git Bash 绝对路径调用：

```powershell
& "C:\Program Files\Git\bin\bash.exe" "D:/reverse_ENV/skill/<name>/scripts/<script>.sh" [参数]
```

Python 脚本 (`.py`)，绝对路径调用：

```powershell
& "D:\reverse_ENV\.venv\Scripts\python.exe" "D:\reverse_ENV\skill\<name>\scripts\<script>.py" [参数]
```

禁止写临时 ad-hoc 命令替代已有脚本。

## APK 逆向脚本

| 脚本 | 路径 | 用途 |
|------|------|------|
| `decode.ps1` | `skill/apk-reverse/scripts/` | 固定项目 jadx/apktool，安全清理生成物，产出 pipeline 摘要 |
| `dump-dex.ps1` | `skill/apk-reverse/scripts/` | LDPlayer/Root + panda whole-DEX 导出，含 ABI/PID/超时/结构校验与失败留证 |
| `frida-run.ps1` | `skill/apk-reverse/scripts/` | Frida 设备/进程管理、spawn/attach；设备列表使用非 TUI Python API |
| `rebuild-sign-install.ps1` | `skill/apk-reverse/scripts/` | 重建 → Android 15 16 KB 对齐检查 → 签名/验签 → adb install |
| `manifest-summary.ps1` | `skill/apk-reverse/scripts/` | 抽取 Manifest 组件/权限 |
| `init-ldplayer-re.ps1` | `skill/apk-reverse/scripts/` | 对指定 LDPlayer ADB serial 验证 Root/ABI/Frida 版本并完成真实 handshake |

## LDPlayer 实例脚本

| 脚本 | 路径 | 用途 |
|------|------|------|
| `re-init.ps1` | `skill/ldplayer-control/scripts/` | 创建/启动项目实例，可用 `-Template re-base|re-xposed|re-stealth` 从模板复制 |
| `re-list.ps1` | `skill/ldplayer-control/scripts/` | 查看 LDPlayer 实例并标注 MAA/RE/NONASCII/ORPHAN |
| `re-proxy.ps1` | `skill/ldplayer-control/scripts/` | 项目实例 HTTPS 代理 on/off，flow 写入 `workspace\<Project>\` |
| `re-backup.ps1` | `skill/ldplayer-control/scripts/` | 将模板或项目实例备份为 `.ldbk` 到 `storage\ldplayer-backups\` |
| `re-restore.ps1` | `skill/ldplayer-control/scripts/` | 从 `.ldbk` 恢复到已有实例，支持 `-SourceProject`；脚本按 index restore 并恢复后重命名，规避备份内部实例名覆盖 |
| `re-destroy.ps1` | `skill/ldplayer-control/scripts/` | 停止实例，可选 `-Remove` 删除 VM，不删除 workspace |


## Web 补环境脚本

| 脚本 | 路径 | 用途 |
|------|------|------|
| `check-isolation.ps1` | `tools\web-env\` | 检查主 Node、RuyiTrace、xbs clone、addon / isolated-vm ABI 状态，确认补环境 runtime 隔离边界 |
| `invoke-xbs-script.ps1` | `tools\web-env\` | 用绝对路径调用 `storage\xbsReverseSkill\web-js-env-patcher\scripts` 下的纯 JS 检查器，临时注入 `RUYI_TRACE_HOME`，不安装依赖、不切主 Node |

调用示例：

```powershell
powershell -File "D:\reverse_ENV\tools\web-env\check-isolation.ps1"
powershell -File "D:\reverse_ENV\tools\web-env\invoke-xbs-script.ps1" -Script "check_tls_clients.js" -ScriptArgs "--markdown"
```
