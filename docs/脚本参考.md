# 脚本参考

## 调用规范

PowerShell 脚本 (`.ps1`)，绝对路径调用：

```powershell
powershell -File "D:\reverse_ENV\skill\<name>\scripts\<script>.ps1" -参数名 值
```

Bash 脚本 (`.sh`)，绝对路径调用：

```bash
bash D:/reverse_ENV/skill/<name>/scripts/<script>.sh [参数]
```

Python 脚本 (`.py`)，绝对路径调用：

```bash
python "D:\reverse_ENV\skill\<name>\scripts\<script>.py" [参数]
```

禁止写临时 ad-hoc 命令替代已有脚本。
